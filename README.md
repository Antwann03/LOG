<h1>Introduction</h1>
<hr><p> This repository contains the VS Code project files for LOG (Lethal Offensive Gear), developed for the Fall 2025 ECE 528: Robotics and Embedded Systems course. The project is built around the ESP32-WROOM-32E microcontroller and utilises DC motor gearboxes, motor drivers, a buck converter, and a custom chassis designed in Fusion 360.
